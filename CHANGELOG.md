## [3.14.1] - 2026-03-02

### 수정됨 (Fixed)
- 일시정지 입력 차단
  - 엔진 `dispatchInput`가 `running === false`일 때 즉시 반환하도록 보강해 일시정지 중 이동/회전 입력 누수를 차단
  - 설정/결과/브리핑뿐 아니라 일반 `pauseOverlay`가 열렸을 때도 게임 입력을 막도록 입력 차단 조건 확장

### 변경됨 (Changed)
- 입력 감각 보정
  - 좌우 방향키 중 하나를 뗄 때 반대 방향키가 이미 눌려 있으면 가장 최근에 유지 중인 키 기준으로 즉시 복귀하도록 키업 보정
  - 스폰 보조(`IHS -> IRS -> inputBuffer`)는 같은 프레임 즉시 실행 구조를 유지하고, 우선순위 변경은 문서 규칙과 충돌해 보류
- 시각 피드백 보강
  - 난이도가 높아질수록 `lineClear` 화면 흔들림 감쇠가 조금 더 빠르게 끝나도록 임팩트 감쇠 프로파일 추가
  - 아이템 블록 글로우를 약 15% 강화해 일반 블록과의 시각적 대비를 높임
